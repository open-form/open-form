# Credits

OpenForm is built on the shoulders of excellent open source projects.

## Core Dependencies

- **[Zod](https://github.com/colinhacks/zod)** — TypeScript-first schema validation
- **[yaml](https://github.com/eemeli/yaml)** — YAML parsing and serialization

## Renderers

- **[docx-templates](https://github.com/guigrpa/docx-templates)** — Template-based DOCX generation
- **[pdf-lib](https://github.com/Hopding/pdf-lib)** — PDF creation and modification
- **[Handlebars](https://github.com/handlebars-lang/handlebars.js)** — Templating engine

## Thank You

We're grateful to the maintainers and contributors of these projects for making OpenForm possible.
