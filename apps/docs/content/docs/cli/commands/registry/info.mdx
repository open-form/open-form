Show detailed information about a registry.

# Usage

```bash
ofm registry info <namespace> [options]
```

# Arguments

| Argument | Description |
|----------|-------------|
| `namespace` | Registry namespace |

# Options

| Option | Description |
|--------|-------------|
| `--json` | Output as JSON |

# Output

```
Acme Forms Registry

Official forms and documents for Acme Corp

URL: https://registry.acme.com
Homepage: https://acme.com/forms
Artifacts: 15

By kind:
  form: 10
  checklist: 3
  document: 2
```
