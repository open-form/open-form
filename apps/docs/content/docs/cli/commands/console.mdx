Opens the OpenForm web console in your default browser.

# Usage

```bash
ofm console
```

# Description

The `console` command provides quick access to the OpenForm web console at `https://open-form.dev`. When executed, it opens the URL in your default web browser.

# Examples

Open the web console:

```bash
ofm console
```

Output:

```
Opening OpenForm console...
```
