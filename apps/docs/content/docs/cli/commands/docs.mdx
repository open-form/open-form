Opens the OpenForm documentation website in your default browser.

# Usage

```bash
ofm docs
```

# Description

The `docs` command provides quick access to the OpenForm documentation. When executed, it opens `https://docs.open-form.dev` in your default web browser.

# Examples

Open documentation:

```bash
ofm docs
```

Output:

```
Opening OpenForm documentation...
```
