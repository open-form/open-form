OpenForm is a framework for defining, validating, and rendering documents as code. Define forms with typed fields, validate user input safely, and render to PDF, DOCX, or HTML.

## Why OpenForm?

- **Type-safe definitions** - Define forms with TypeScript, get full type inference
- **Built-in validation** - Zod-style parsing with detailed error messages
- **Multi-format output** - Same data renders to PDF, DOCX, HTML, or plain text
- **Version control** - YAML/JSON artifacts work with git like source code
- **No vendor lock-in** - Open standards, MIT licensed, works offline

## Quick Example

```typescript
import { open } from "@open-form/sdk";
import { textRenderer } from "@open-form/sdk";

// Define a form
const form = open
  .form()
  .name("contact")
  .version("1.0.0")
  .title("Contact Form")
  .fields({
    name: { type: "text", required: true },
    email: { type: "email", required: true },
  })
  .inlineTemplate("html", "text/html", "<p>{{name}}: {{email}}</p>")
  .defaultTemplate("html")
  .build();

// Validate data
const result = form.safeParseData({
  name: "Alice",
  email: "alice@example.com",
});

if (result.success) {
  // Render to HTML
  const html = await form.render(textRenderer, { data: result.data });
}
```
