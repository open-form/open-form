OpenForm defines shared types that are used across multiple artifact types.

| Type                                         | Description                                              |
| -------------------------------------------- | -------------------------------------------------------- |
| [Layer](/schemas/artifacts/shared/layer)     | Rendering output specification (inline or file-backed)   |
| [Logic](/schemas/artifacts/shared/logic)     | Named expressions and conditional logic                  |
