---
title: Organization
description: An organization's identity and legal details
---

An organization's identity including legal name, entity type, and tax information.

## Properties

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| `name` | `string` | Yes | Common or trade name |
| `legalName` | `string` | No | Registered/legal name |
| `domicile` | `string` | No | Country/region of domicile |
| `entityType` | `string` | No | Legal entity type (corporation, LLC, etc.) |
| `entityId` | `string` | No | Business identification number |
| `taxId` | `string` | No | Tax identification number |

## Examples



```yaml tab="YAML"
name: Acme Corp
legalName: Acme Corporation, Inc.
domicile: United States
entityType: corporation
entityId: "12345678"
taxId: "12-3456789"
```


```json tab="JSON"
{
  "name": "Acme Corp",
  "legalName": "Acme Corporation, Inc.",
  "domicile": "United States",
  "entityType": "corporation",
  "entityId": "12345678",
  "taxId": "12-3456789"
}
```
