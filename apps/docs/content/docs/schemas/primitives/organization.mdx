title: Organization

An Organization is a structured primitive that represents a legal entity, such as a company, agency, or organization.

    isOptional: true,

  {
    name: 'entityType',

    isOptional: true,

  {

    name: 'taxId',

  },
]} />

# Examples



```yaml tab="YAML"
name: Acme Corp
legalName: Acme Corporation, Inc.
domicile: United States
entityType: corporation
entityId: "12345678"
taxId: "12-3456789"
```
