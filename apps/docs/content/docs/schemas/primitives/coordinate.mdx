---
title: Coordinate
description: A geographic coordinate
---

A geographic coordinate with latitude and longitude in decimal degrees (WGS84).

## Properties

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| `lat` | `number` | Yes | Latitude in decimal degrees, range -90 to 90 |
| `lon` | `number` | Yes | Longitude in decimal degrees, range -180 to 180 |

## Examples



```yaml tab="YAML"
# San Francisco, CA
lat: 37.7749
lon: -122.4194
```


```json tab="JSON"
{
  "lat": 37.7749,
  "lon": -122.4194
}
```
