---
title: Person
description: A person's name
---

A person's name with a required full name and optional components (title, first, middle, last, suffix).

## Properties

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| `fullName` | `string` | Yes | Full name as a single string |
| `title` | `string` | No | Prefix or honorific (Mr., Dr., etc.) |
| `firstName` | `string` | No | First/given name |
| `middleName` | `string` | No | Middle name or initial |
| `lastName` | `string` | No | Last/family name |
| `suffix` | `string` | No | Suffix (Jr., Sr., III, etc.) |

## Examples



```yaml tab="YAML"
fullName: Dr. Jane Marie Smith, Ph.D.
title: Dr.
firstName: Jane
middleName: Marie
lastName: Smith
suffix: Ph.D.
```


```json tab="JSON"
{
  "fullName": "Dr. Jane Marie Smith, Ph.D.",
  "title": "Dr.",
  "firstName": "Jane",
  "middleName": "Marie",
  "lastName": "Smith",
  "suffix": "Ph.D."
}
```



### Minimal

```yaml tab="YAML"
fullName: John Doe
```

```json tab="JSON"
{
  "fullName": "John Doe"
}
```
