A Person is a structured primitive that represents a natural person.

# Examples

```yaml tab="YAML"
fullName: Dr. Jane Marie Smith, Ph.D.
title: Dr.
firstName: Jane
middleName: Marie
lastName: Smith
suffix: Ph.D.
```
