---
title: Blocks
description: Composable pieces within artifacts
---

Blocks are the composable pieces that make up artifacts. They define the structure within forms and other artifact types.

## Block Types

| Block | Description |
|-------|-------------|
| [Fields](/docs/schemas/blocks/fields) | Field types for capturing data |
| [Fieldset](/docs/schemas/blocks/fieldset) | Logical grouping of fields |
| [Annex](/docs/schemas/blocks/annex) | Slots for supplementary documents |
| [Layers](/docs/schemas/blocks/layers) | Inline and file-based layer definitions |
| [Logic](/docs/schemas/blocks/logic) | Conditional expressions |
