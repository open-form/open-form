---
title: Layers
description: Inline and file-based layer definitions
---

A Layer defines how an artifact renders into a specific format. Layers can be inline (embedded text) or file-backed (external file reference).

## Layer Types

A Layer can be either an InlineLayer or a FileLayer.

## InlineLayer

Inline layers embed the content directly in the artifact definition.

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| `kind` | `'inline'` | Yes | Literal "inline" discriminator |
| `mimeType` | `string` | Yes | MIME type of the content (e.g., text/markdown, text/html) |
| `text` | `string` | Yes | Layer content with interpolation placeholders |
| `title` | `string` | No | Human-readable title for this layer |
| `description` | `string` | No | Description of what this layer represents |
| `checksum` | `string` | No | SHA-256 checksum for integrity verification |
| `bindings` | `Record<string, string>` | No | Field bindings for the layer (typically for PDF) |

## FileLayer

File layers reference an external file for the layer content.

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| `kind` | `'file'` | Yes | Literal "file" discriminator |
| `mimeType` | `string` | Yes | MIME type of the file (e.g., application/pdf) |
| `path` | `string` | Yes | Absolute path from repo root to the layer file |
| `title` | `string` | No | Human-readable title for this layer |
| `description` | `string` | No | Description of what this layer represents |
| `checksum` | `string` | No | SHA-256 checksum for integrity verification |
| `bindings` | `Record<string, string>` | No | Field bindings for the layer (typically for PDF) |

## Examples

### Inline Layer

```yaml tab="YAML"
layers:
  markdown:
    kind: inline
    mimeType: text/markdown
    title: Markdown Summary
    text: |
      # {{title}}

      **Applicant:** {{fields.firstName}} {{fields.lastName}}
      **Email:** {{fields.email}}
```

```json tab="JSON"
{
  "layers": {
    "markdown": {
      "kind": "inline",
      "mimeType": "text/markdown",
      "title": "Markdown Summary",
      "text": "# {{title}}\n\n**Applicant:** {{fields.firstName}} {{fields.lastName}}\n**Email:** {{fields.email}}"
    }
  }
}
```

### File Layer

```yaml tab="YAML"
layers:
  pdf:
    kind: file
    mimeType: application/pdf
    path: /templates/application-form.pdf
    title: PDF Application
    bindings:
      full_name: firstName
      email_address: email
      submit_date: submittedAt
```

```json tab="JSON"
{
  "layers": {
    "pdf": {
      "kind": "file",
      "mimeType": "application/pdf",
      "path": "/templates/application-form.pdf",
      "title": "PDF Application",
      "bindings": {
        "full_name": "firstName",
        "email_address": "email",
        "submit_date": "submittedAt"
      }
    }
  }
}
```

## Common MIME Types

| MIME Type | Description |
|-----------|-------------|
| `text/markdown` | Markdown text |
| `text/html` | HTML document |
| `text/plain` | Plain text |
| `application/pdf` | PDF document |
| `application/json` | JSON data |
