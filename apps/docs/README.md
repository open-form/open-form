# @open-form/docs

> OpenForm documentation site

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Documentation website for the OpenForm framework, available at [docs.open-form.dev](https://docs.open-form.dev).

## Tech Stack

- [Fumadocs](https://fumadocs.vercel.app/) - MDX documentation framework
- [TanStack Start](https://tanstack.com/start) - Full-stack React framework
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [Vite](https://vitejs.dev/) - Build tool

## Development

```bash
# Start development server
pnpm dev

# Build for production
pnpm build

# Type check
pnpm check-types
```

The documentation server runs on port 3001 by default.

## Content

Documentation content is located in the `content/` directory as MDX files.
